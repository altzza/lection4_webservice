* Компилируем
    ```shell
    javac  Server
    ```
* Компилируем 
    ```shell
    javac  Client
    ```
* Запускаем сервер
    ```shell
    java Server
    ```

* а потом, дождавшись надписи "server is started", и любое количество клиентов:
    ```shell
    java Client test1
    java Client test2
    ...
    java Client testN
    ```
